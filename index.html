🏗 About SiteSpy

SiteSpy is a mobile-first construction project records and estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

👤 My Contribution

I owned the Firestore and Firebase Storage security rules for SiteSpy, along with the documentation that explains how and why they work. firestore.rules defines who can read and write to each collection in the database. storage.rules does the same for uploaded files. docs/Firebase_Setup_Guide.md explains how to configure the entire Firebase project from scratch so the security layer is correctly in place before any data is stored.

💡 Reflection

Security rules are not optional — they are the last line of defence between user data and the internet. I learned to think adversarially: what would a malicious request look like, and does my rule block it? Writing the setup guide forced me to understand every configuration step deeply enough to explain it to someone who has never used Firebase. That level of understanding made me significantly more confident in the rules I wrote.
